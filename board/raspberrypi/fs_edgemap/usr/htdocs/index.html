Welcome to Edgemap!

This Edgemap hosts full world OSM map. It works with Meshtastic LoRA radios and can be configured to various other communication methods like CoT sources etc. Edgemap is usable with most web browsers without any additional application installs.

• To access Edgemap Web UI - download CA certificate and install it to your system.

• Excalidraw is locally hosted whiteboard

• Cryptpad is locally hosted collaboration suite

• The Lounge is browser based IRC client

For more information, checkout Resilience Theatre Wiki page and project Github.
